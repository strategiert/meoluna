# Meoluna TikTok Scripts

## Format-Regeln
- **Länge:** 15-60 Sekunden
- **Hook:** Erste 2 Sekunden entscheiden
- **Untertitel:** Immer (80% schauen ohne Ton)
- **CTA:** Soft, nicht salesy

---

## Script 1: Einmaleins-Trick (Educational)

**Hook:** "Einmaleins in 5 Sekunden?"

**Script:**
```
[Zeigt Hände]
"9 mal 7 — wie schnell kannst du das?"

[Pause]

"Halte 10 Finger hoch."
[Zeigt 10 Finger]

"Klappe den 7. Finger um."
[Klappt Finger 7 um]

"Links der Klappe — das sind die Zehner. 6."
[Zeigt 6 Finger links]

"Rechts der Klappe — das sind die Einer. 3."
[Zeigt 3 Finger rechts]

"63. Fertig."

[Text einblenden: "Funktioniert für ALLE 9er-Aufgaben"]

"Speichern und an ein Kind weiterschicken, das Mathe hasst."
```

**Sounds:** Trending sound oder Original

## Script 2: POV Eltern (Relatable)

**Hook:** "POV: Dein Kind sagt 'Ich hasse Mathe'"

**Script:**
```
[Gesicht zeigen, genervter Ausdruck]
"Schon wieder."

[Cut]
"Und du weißt: Du warst auch nicht gut in Mathe."

[Cut]
"Aber irgendwer muss helfen."

[Cut, Bildschirm zeigen]
"Also probierst du Apps."

[Schnelle Cuts von verschiedenen "schlechten" Apps]
"Zu viel Werbung."
"Zu teuer."
"Macht keinen Spaß."

[Cut, Meoluna öffnen]
"Bis du DAS hier findest."

[Zeigt Meoluna, Kind macht Übung]
"Und dein Kind fragt plötzlich: 'Kann ich noch eine Runde?'"

[Gesicht, überrascht]
"Wait what?"

[Text: "Link in Bio"]
```

## Script 3: Fun Fact (Edutainment)

**Hook:** "Fakt, den 99% nicht wissen:"

**Script:**
```
"47% der deutschen Viertklässler können nicht richtig lesen."

[Pause für Schock]

"Fast. Die. Hälfte."

"Und wir diskutieren, ob iPads in Schulen gehören?"

[Kopfschütteln]

"Das Problem ist nicht Technologie."

"Das Problem ist, dass wir Kindern beibringen, Lernen sei langweilig."

[Zeigt Meoluna kurz]

"Was wenn es anders geht?"

"Folgen für mehr Education Content."
```

## Script 4: Stitch/Duett Vorlage

**Für Stitches auf "Schule ist sinnlos" o.ä. Videos:**

```
[Erklärt kurz alternatives Lernen]

A) 2 Stunden Arbeitsblätter ausfüllen
B) 20 Minuten interaktives Spiel, das dasselbe lehrt

Was würden sie wählen?"

"Genau."
```

## Script 5: Quick Hack (15 Sek)

**Hook:** "Mathe-Hack für Eltern"

**Script:**
```
"Dein Kind lernt NICHT durch Wiederholen."

"Es lernt durch VERSTEHEN."

"Frag nicht: 'Was ist 7 mal 8?'"

"Frag: 'Wie kommst du auf die Antwort?'"

"Game changer."

[Text: Mehr Lerntipps → Folgen]
```

## Script 6: Transformation (Before/After)

**Hook:** "Montag vs. Freitag"

**Script:**
```
[Split screen oder Transition]

MONTAG:
- Kind: "Ich kann das nicht!"
- Tränen
- Hausaufgaben dauern 2 Stunden
- Eltern frustriert

[Transition-Sound]

FREITAG (nach 1 Woche Meoluna):
- Kind: "Noch eine Runde!"
- Lacht
- 20 Minuten konzentriertes Üben
- Eltern chillen

[Text: Link in Bio]
```

## Engagement-Taktik

1. **Kommentare:** Auf JEDEN Kommentar antworten (erste 30 Min)
2. **Fragen stellen:** "Welchen Trick soll ich als nächstes zeigen?"
3. **Stitches nutzen:** Auf Education-Content reagieren
4. **Trends mitmachen:** Aber immer Education-Twist

---

## Don'ts

❌ Zu werblich ("Kauf unsere App!")
❌ Zu lang (>60 Sek ohne sehr guten Grund)
❌ Schlechte Audio-Qualität
❌ Langweilige Hooks
❌ Keine Untertitel
